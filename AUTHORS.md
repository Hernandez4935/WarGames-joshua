## Contribution Types

We recognize and appreciate various types of contributions:

### Code Contributors
People who have contributed code to the project through pull requests.

### Documentation Contributors
People who have improved documentation, including README files, guides, rustdoc comments, and planning documents.

### Design Contributors
People who have contributed to architecture design, UI/UX design, or system design.

### Testing Contributors
People who have added tests, improved test coverage, or performed manual testing and bug reporting.

### Security Researchers
People who have responsibly disclosed security vulnerabilities.

### Issue Reporters
People who have reported bugs, suggested features, or participated in discussions.

### Reviewers
People who have reviewed pull requests and provided valuable feedback.

## How to Contribute

We welcome contributions from everyone! Here's how to get started:

1. **Read the Contributing Guidelines**: See [CONTRIBUTING.md](CONTRIBUTING.md)
2. **Check the Code of Conduct**: See [CODE_OF_CONDUCT.md](CODE_OF_CONDUCT.md)
3. **Find an Issue**: Look for issues labeled `good first issue` or `help wanted`
4. **Submit a Pull Request**: Follow our PR template and guidelines
5. **Join Discussions**: Participate in GitHub Discussions

## Recognition

Contributors are recognized in several ways:

- **Listed in this file** - All contributors are listed here
- **Git commit history** - Your authorship is preserved in version control
- **Release notes** - Significant contributions mentioned in CHANGELOG.md
- **GitHub contributors page** - Automatic recognition on the repository

## Adding Yourself to This List

If you've contributed to the project and would like to be listed here:

1. Fork the repository
2. Add your information to the appropriate section
3. Submit a pull request with the title "docs: add [Your Name] to AUTHORS.md"
4. Include a brief description of your contributions

**Format:**
```markdown
- **Your Name** ([@username](https://github.com/username))
  - Description of contributions
  - Optional: Contact information
```
